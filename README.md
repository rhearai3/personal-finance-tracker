## Overview

This project helps you track your daily financial transactions in one place.  
It supports:
- Adding income and expense entries
- Importing data automatically from **Google Sheets**
- Storing transactions securely in a **PostgreSQL** database
- Viewing transaction history interactively
- Gaining insights with **bar and pie charts** (via Plotly)

Built with:
- Python
- Streamlit
- PostgreSQL + SQLAlchemy
- Plotly
- Google Sheets API (gspread)

---

## Features

Add transactions manually  
Import from Google Sheets  
Categorize and analyze expenses  
Interactive dashboards (Income vs Expense, Category Distribution)  
Persistent database storage (PostgreSQL)  
Secure credential management (via `.gitignore` and environment variables)
